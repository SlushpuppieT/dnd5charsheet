About this sheet

A local-first D&D 5e character sheet. Your character is auto-saved in this browser and can be downloaded as JSON with Save. Pull SRD class/race/background/feat/spell data from open5e to auto-fill choices and links.

Quick tips

  • Rest opens the rest dialog — Short rolls Hit Dice for healing, Long restores everything.
  • Click a spell name's ↗ to open its rules (with 2014 / 2024 / A5E tabs when multiple editions exist).
  • Ability scores show your base in the small box and the final below the modifier; +R / +L / +F tags break down where each bonus came from.
  • Pick a class to auto-populate hit dice, save proficiencies, spellcasting ability, and class-resource pools.
  • Offline: the selected class / race / background are cached on the sheet, so class features, spell-slot tables, expected saves, and hit-die deviation keep rendering without a connection. Picking a new class / race / background needs the SRD API to be reachable.

Non-standard highlight

Fields that differ from your class's defaults get a subtle amber highlight. It's not an error — just a visual cue that you've gone off-spec. Hover the highlighted element for a tooltip explaining the deviation. Watched fields:

  • Hit Die picker (e.g. Bard normally d8 — picking d12 flags it).
  • Saving Throw checkbox (a save proficiency the class doesn't grant, or missing one it does).
  • Skill checkbox (proficient without a class/race/background source, or an auto-granted skill that's been unchecked).
  • Cantrips picked / Spells picked counter when the count exceeds Known + Bonus.
  • Feats counter when you've added more feats than your ASI + Race slots provide.

Subclass Supplement

The SRD only contains one subclass per class (Circle of the Land, Champion, Thief, etc.). Other PHB subclasses (Circle of the Moon, Battle Master, Beast Master, etc.) aren't legally redistributable, so they aren't in any public API. You can load a community-curated supplement here.

  1. Download the starter file (PHB subclass names + levels + placeholders).
  2. Open it in a text editor and fill in the desc text for each feature from your own copy of the PHB. The README at the bottom of the file documents the schema (skills, resource pools, etc.).
  3. Upload the filled-in JSON back here. The subclasses appear in the Subclass dropdown for the matching class.

Race Supplement

The SRD only contains one subrace per race (High Elf, Hill Dwarf, Lightfoot Halfling, Rock Gnome). Use this section to add missing PHB races/subraces like Wood Elf, Drow, Mountain Dwarf, Stout Halfling, Forest Gnome, or homebrew races.

Data sources

SRD content is pulled live from open5e.com, an aggregator of legally redistributable 5e material from these publishers:

  • Open5e aggregator + API
  • WotC SRD 5.1 (2014) wotc-srd / srd-2014
  • WotC SRD 5.2 (2024) srd-2024
  • Level Up: Advanced 5e a5e / a5e-ag — EN Publishing
  • Kobold Press Tome of Heroes (toh), Kobold Press (kp), Tome of Beasts (tob)
  • Black Flag Roleplaying blackflag
  • Darrington Press Critical Role: Tal'Dorei (taldorei)
  • Open5e Original Content o5e
