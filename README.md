# D&D 5e Character Sheet

A local-first, mobile-friendly D&D 5th Edition character sheet that runs entirely in the browser — no account, no server, no install required.

**[▶ Open the sheet](https://slushpuppiet.github.io/dnd5charsheet/)**

---

## Features

- **Local-first** — your character auto-saves in the browser; nothing is sent to a server
- **Save / Load** — export and import characters as JSON files
- **SRD data** — class features, spells, races, backgrounds, feats, and spell rules pulled live from [Open5e](https://open5e.com)
- **Mobile-friendly** — bottom tab bar, hamburger menu, and compact layout for phone use
- **Multiple themes** — dark, light, and several colour variants
- **Supplement system** — load custom subclass or race JSON files to add content beyond the SRD
- **Markup mode** — rich formatting (headings, bold, bullets, links) in long text fields
- **Offline capable** — once a class/race/background is selected, core features render without a connection

## Usage

Just open `index.html` in any modern browser. No build step, no dependencies.

For the hosted version, visit the link above.

## Supplement Files

The `Supplements/` folder contains starter JSON files for adding non-SRD content:

- `subclass-supplement-starter.json` — PHB subclass names and level placeholders; fill in feature text from your own PHB copy
- `race-supplement-starter.json` — starter template for additional races and subraces

Load these via the **ⓘ Info** panel inside the sheet.
